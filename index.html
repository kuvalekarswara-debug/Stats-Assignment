What is a Distribution?

A probability distribution describes how probabilities are assigned to all possible values of a random variable. It completely characterizes the random variable's behavior.

üé≤

Discrete Distributions

  • Bernoulli: Single trial with success/failure (X ‚àà {0,1})
  • Binomial: Number of successes in n trials
  • Poisson: Number of events in fixed time/space
  • Geometric: Trials until first success
üìà

Continuous Distributions

  • Uniform: Equal probability over interval [a, b]
  • Normal (Gaussian): Bell curve, most common in nature
  • Exponential: Time between events (memoryless)
  • Beta: Probabilities and proportions
üîç

Cumulative Distribution Function (CDF)

For both discrete and continuous RVs, the CDF gives the probability that X is less than or equal to a value:

  • limx‚Üí-‚àû F(x) = 0

  • F(x) is right-continuous

Challenge Yourself

Test your understanding with these thought-provoking questions
1

Why is P(X = a) = 0 for continuous random variables, yet we can still observe specific values?

Answer: This is a fundamental concept in measure theory. For continuous RVs, there are uncountably infinite possible values, so the probability of hitting any exact single point is infinitesimally small (mathematically zero). However, we observe specific values because:

  • Real measurements have finite precision (e.g., 170.5 cm actually represents an interval [170.45, 170.55))
  • The PDF f(x) represents probability density, not probability itself
  • Probability exists only over intervals: P(a ‚â§ X ‚â§ b) = ‚à´·µÉ·µá f(x) dx > 0
  • The zero probability means it's theoretically impossible to hit an exact real number with infinite precision
2

Can a random variable be both discrete and continuous? What about neither?

Answer: A random variable cannot be both discrete and continuous simultaneously (they are mutually exclusive by definition). However, a random variable can be neither - this is called a mixed random variable.

Example: Insurance claim amount X where:

  • P(X = 0) = 0.7 (70% chance of no claim - discrete component)
  • If X > 0, X follows a continuous distribution (e.g., exponential for claim amounts)

This has both a probability mass at X=0 and a continuous density for X>0, making it a mixed distribution.

3

If two random variables have the same expected value and variance, are they necessarily the same distribution?

Answer: No! Expected value and variance only capture the first two moments of a distribution. Many different distributions can share the same mean and variance.

Counterexample:

  • Let X‚ÇÅ = {-1, 0, 1} with P(X‚ÇÅ = -1) = 0.25, P(X‚ÇÅ = 0) = 0.5, P(X‚ÇÅ = 1) = 0.25
  • Let X‚ÇÇ = {-2, 2} with P(X‚ÇÇ = -2) = 0.5, P(X‚ÇÇ = 2) = 0.5

Both have E[X] = 0 and Var(X) = 1, but completely different distributions! To fully characterize a distribution, you need all moments, or the entire PMF/PDF, or the moment-generating function.

4

Consider tossing a fair coin until you get a head. Is the sample space finite or infinite? What about the random variable X = "number of tosses"?

Answer: Both the sample space and the random variable are countably infinite.

Sample Space: S = {H, TH, TTH, TTTH, ...} - infinite but listable sequences

Random Variable: X ‚àà {1, 2, 3, 4, ...} follows a geometric distribution

PMF: P(X = k) = (1/2)^k for k = 1, 2, 3, ...

Key Insight: Even though there's no upper bound on the number of tosses (theoretically could flip tails forever), each specific outcome has a positive probability that decreases geometrically. The sum Œ£ P(X=k) = 1, confirming this is a valid probability distribution despite being infinite.

5

Why do we use different notation for discrete (PMF: p(x)) vs continuous (PDF: f(x)) random variables? What goes wrong if we try to use the same approach?

Answer: The notation reflects a fundamental mathematical difference:

Discrete (PMF): p(x) = P(X = x) directly gives probability

  • Sum of all p(x) = 1
  • Can ask "what's the probability of exactly x?"

Continuous (PDF): f(x) is probability density, not probability

  • f(x) can be > 1 (it's a density!)
  • ‚à´ f(x) dx = 1 (integral, not sum)
  • P(X = x) = 0 for any specific x
  • P(a ‚â§ X ‚â§ b) = ‚à´·µÉ·µá f(x) dx

What goes wrong: If we tried to assign probabilities P(X = x) to each point in a continuous distribution, we'd have uncountably many points, each needing probability > 0, making the total infinite (violating Œ£ P = 1). Density functions solve this by using integration over intervals instead.

6

A biased coin lands heads with probability p. You flip it n times. What is the probability that the number of heads equals the number of tails? How does this relate to discrete random variables?

Answer: Let X = number of heads in n flips. X follows Binomial(n, p).

For equal heads and tails: We need X = n/2, which requires n to be even!

  • If n is odd: P(X = n/2) = 0 (impossible to have equal splits)
  • If n is even: P(X = n/2) = C(n, n/2) ¬∑ p^(n/2) ¬∑ (1-p)^(n/2)

Special cases:

  • Fair coin (p = 0.5): P(X = n/2) = C(n, n/2) / 2^n
  • As n ‚Üí ‚àû: This probability decreases for any fixed p

Connection to RVs: This demonstrates how discrete RVs assign exact probabilities to specific outcomes, and how the sample space structure (odd vs even n) affects possible values of the random variable.

7

Can you construct a sample space where the sample space itself is uncountably infinite, but the random variable defined on it is discrete? Give an example.

Answer: Yes! The nature of the sample space doesn't determine whether the random variable is discrete or continuous.

Example: Dart throw on a continuous circular dartboard

  • Sample Space S: All points (x, y) on the unit circle - uncountably infinite
  • Random Variable X: Score based on which ring the dart lands in:
    • X = 10 if distance from center r ‚â§ 0.2
    • X = 5 if 0.2 < r ‚â§ 0.6
    • X = 1 if 0.6 < r ‚â§ 1.0

Here S is continuous (uncountably many points), but X only takes values {1, 5, 10}, making it discrete! This shows that discreteness/continuity is a property of the random variable's range, not the sample space itself.
